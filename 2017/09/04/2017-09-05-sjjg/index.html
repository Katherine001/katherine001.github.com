设计顺序串上实现串比较运算Strcmp(s,t)的算法

[分析]

算法思路如下：

（1）比较s和t两个串的共同长度范围内的对应字符
Ⅰ.若s的字符>t的字符，返回1
Ⅱ.若s的字符<t的字符，返回-1
Ⅲ.若s的字符=t的字符，按上述规则继续比较
（2）当（1）中对应字符均相同时，比较s和t的长度
Ⅰ.两者相等时，返回 0
Ⅱ.s长度>t长度，返回1
Ⅲ.s长度<t长度，返回-1
