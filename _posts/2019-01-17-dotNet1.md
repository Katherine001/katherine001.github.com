# 一、.net程序基本编写、执行流程

- 编写C#代码，保存为.cs文件
- 通过csc.exe程序来将.cs文件编译为.net程序集（.exe或者.dll）。此时的exe或dll并不是机器码（cpu不可理解）。
- 程序运行时通过JIT（Just In Time）即时编译，将程序集编译为cpu能理解的机器码，这是cpu才能执行。（这个编译过程会和当前机器有关（根据当前机器的内存和cpu等））。

## （三）类、对象

### 1.什么是类  

类是模具，创建对象的模具，**抽象的**：

- 类是一种数据类型，用户自定义的数据类型
- 类组成：
  - **字段**
  - **属性**
  - **方法**
  - **构造函数**
  - ……

### 2.什么是对象

对象是具体的，是类的具体实例。对象具有**属性**（特征）和**方法**（行为）

### 3.this关键字

代表当前对象

### 4.案例：老师、学生类

#### （1）写学生类:

```c#
public class Student
{
    //定义好一个类后，不写构造函数会有一个默认的无参构造函数
    //添加构造函数，当为类手动编写一个构造函数后，会覆盖默认的那个构造函数
    public Student()
    {
        
    }
    //重载构造函数
    public Student(string name,int age,string gender,string sid)
    {
        this.Name = name;
        this.Age = age;
        this.Gender = gender;
        this.Sid = sid;
    }
    private string name;
    private int age;
    private string gender;
    private string sid;
    public string Name
    {
        get{return name;}
        set{name = value;}
    }
    public int Age
    {
        get{return age;}
        set{age = value;}
    }
    public string Gender
    {
        get{return gender;}
        set{gender = value;}
    }
    public string Sid
    {
       get{return sid;}
       set{sid = value;}
    }
}
```

### 5.案例：迷你计算器

#### （1）定义计算器类

```c#
namespace 迷你计算器
{
    public class Calculator
    {
        public Calculator(double d1,double d2)
        {
            this.Number1 = d1;
            this.Number2 = d2;
        }
        public double Number1
        {
            get;
            set;
        }
        public double Number2
        {
            get;
            set;
        }
        //加法
        public double Add()
        {
            return Number1 + Number2;
        }
        //减法
        public double Cut()
        {
            return Number1 - Number2;
        }
        //乘法
        public double Mul()
        {
            return Number1 * Number2;
        }
        //除法
        public double Div()
        {
            return Number1 / Number2;
        }
    }
}
```

### 6.案例：猜拳游戏

**需求：用户在界面上点击相应的按钮选择出拳，电脑随机产生出拳，然后判定结果并显示在屏幕上**

#### （1）定义玩家类

```c#
namespace 猜拳游戏
{
    public class User
    {
        public string FistName
        {
            get;
            set;
        }
        //玩家的出拳方法
        public int showFist(string fist)
        {
            this.FistName = fist;
            int result = -1;
            switch(fist)
            {
                case "石头":
                    result = 1;
                    break;
                case "剪刀":
                    result = 2;
                    break;
                case "布":
                    result = 3;
                    break;
            }
            return result;
        }
    }
}
```

#### （2）定义电脑玩家类

```c#
namespace 猜拳游戏
{
    public class Computer
    {
        //用来保存计算机出拳结果的属性
        public string FistName
        {
            get;
            set;
        }
        //计算机的出拳方法
        public int showFist()
        {
            Random random = new Random();
            int r = random.Next(1, 4);
            //1.石头
            //2.剪刀
            //3.布
            switch(r)
            {
                case 1:
                    this.FistName = "石头";
                    break;
                case 2:
                    this.FistName = "剪刀";
                    break;
                case 3:
                    this.FistName = "布";
                    break;
            }
            return r;
        }
    }
}
```

#### （3）定义裁判类

```c#
namespace 猜拳游戏
{
    public class CaiPan
    {
        public string isUserWin(int user,int computer)
        {
            if(user - computer == 0)
            {
                return "平局";
            }
            else if(user - computer == -1 || user-computer == 2)
            {
                return "用户赢了";
            }
            else
            {
                return "输了";
            }
        }
    }
}
```
